FAQ

Common Questions

Every project starts with discovery covering business goals, user needs, and constraints. I move through research, IA, wireframes, high-fidelity UI, prototyping, usability testing — all with WCAG compliance — before clean developer handoff.
I establish baseline metrics before redesign — conversion rates, bounce rates, task completion. Post-launch, I recommend A/B testing. Typical results include 30–200% conversion improvement and 25–40% drop-off reduction.
Accessibility is designed in, not bolted on. All colours pass WCAG 2.1 AA contrast ratios. I annotate wireframes with semantic HTML structure, ARIA labels, and keyboard navigation patterns. Handoff includes accessibility checklist.
UX Audit: 1–2 weeks. Full Product Redesign: 4–8 weeks depending on scope. Design Partnership retainers run on 2-week sprint cycles.
Figma for UI design, prototyping, and design systems. Framer for motion prototypes, Maze for research, Adobe Creative Suite for graphics. Handoffs include Figma Inspect and design tokens.
